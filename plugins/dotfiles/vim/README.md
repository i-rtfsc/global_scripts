# Vim Configuration

请将你的 Vim 配置文件放置在此目录下，命名为 `.vimrc`

## 安装

```bash
gs dotfiles vim install
```

## 配置文件路径

- 源文件: `plugins/dotfiles/vim/.vimrc`
- 目标文件: `~/.vimrc`

## 其他命令

- `gs dotfiles vim backup` - 备份当前配置
- `gs dotfiles vim restore` - 恢复备份
- `gs dotfiles vim uninstall` - 卸载配置
- `gs dotfiles vim status` - 查看状态
