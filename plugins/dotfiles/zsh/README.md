# Zsh Configuration

请将你的 Zsh 配置文件放置在此目录下，命名为 `.zshrc`

## 安装

```bash
gs dotfiles zsh install
```

## 配置文件路径

- 源文件: `plugins/dotfiles/zsh/.zshrc`
- 目标文件: `~/.zshrc`

## 其他命令

- `gs dotfiles zsh backup` - 备份当前配置
- `gs dotfiles zsh restore` - 恢复备份
- `gs dotfiles zsh uninstall` - 卸载配置
- `gs dotfiles zsh status` - 查看状态
