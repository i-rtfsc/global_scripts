# Tmux Configuration

请将你的 Tmux 配置文件放置在此目录下，命名为 `.tmux.conf`

## 安装

```bash
gs dotfiles tmux install
```

## 配置文件路径

- 源文件: `plugins/dotfiles/tmux/.tmux.conf`
- 目标文件: `~/.tmux.conf`

## 其他命令

- `gs dotfiles tmux backup` - 备份当前配置
- `gs dotfiles tmux restore` - 恢复备份
- `gs dotfiles tmux uninstall` - 卸载配置
- `gs dotfiles tmux status` - 查看状态
