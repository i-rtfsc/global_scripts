# Git Configuration

请将你的 Git 配置文件放置在此目录下，命名为 `.gitconfig`

## 安装

```bash
gs dotfiles git install
```

## 配置文件路径

- 源文件: `plugins/dotfiles/git/.gitconfig`
- 目标文件: `~/.gitconfig`

## 其他命令

- `gs dotfiles git backup` - 备份当前配置
- `gs dotfiles git restore` - 恢复备份
- `gs dotfiles git uninstall` - 卸载配置
- `gs dotfiles git status` - 查看状态
